# Contributing

Thanks for contributing to `pytest-mellea`.

## Setup

```bash
mise run uv:sync
```

This installs Python and `uv` through `mise`, syncs dependencies, and writes native git hooks.

## Checks

```bash
mise run lint:fix
mise run lint:all
mise run tests:run
mise run release:build
```

Do not add `.pre-commit-config.yaml` or the `pre-commit` Python package. Git hooks are generated by `mise.toml`.

## Commit style

Use Angular-style commit prefixes such as `feat:`, `fix:`, `docs:`, `test:`, and `refactor:`.
